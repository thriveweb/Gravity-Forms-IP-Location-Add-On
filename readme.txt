A simple plugin that replaces {user:country} merge tag with the country of the user's IP.

== Description ==

Gravity Forms IP Location Add-On is a straightforward and easy-to-use plugin that adds location functionality to your Gravity Forms. This plugin introduces a new merge tag: `{user:country}`, which gets replaced with the country name associated with the user's IP address.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/Gravity-Forms-IP-Location-Add-On` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the IPLocation Settings->IPLocation Settings screen to configure the plugin and add your IPStack access key.
4. Use `{user:country}` merge tag in your Gravity Forms.

== Frequently Asked Questions ==

= Where do I get an IPStack access key? =

You can obtain an access key by creating an account on the IPStack website.
